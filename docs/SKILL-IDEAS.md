# Claude Code Skill Ideas (Developer-Focused)

Skill ideas for Apple platform and general development workflows.

> **Note**: For life/work skills (freelancing, finance, health, productivity), see [claude-code-life-skills](https://github.com/rshankras/claude-code-life-skills).

---

## Developer Tools

| Skill | Purpose |
|-------|---------|
| **pr-description-writer** | Generate PR descriptions from git diff |
| **api-docs-generator** | Generate API documentation from code |
| **regex-builder** | Build and explain regex patterns |
| **sql-query-helper** | Help write/optimize SQL queries |
| **git-workflow** | Guide through complex git operations (rebase, cherry-pick) |
| **changelog-writer** | Write engaging release notes from commits |
| **code-explainer** | Explain complex code blocks in plain English |
| **refactor-advisor** | Suggest refactoring opportunities |

## App Store & Release

| Skill | Purpose |

| **review-response-writer** | Write professional responses to app reviews |
| **release-notes-writer** | Write user-friendly release notes |
| **privacy-policy-generator** | Generate privacy policy for apps |
